**HyDB2**는 하이포스 프로그램이 설치된 서버 환경에서 **주기적 또는 즉시 백업**을 수행하고,  
FTP 업로드 및 자동 종료 등 부가 기능을 제공하는 **데이터베이스 백업 도구**입니다.  

주 목적은 **매장 데이터 즉 데이터베이스의 안전한 보관과 원격지 업로드**를 통한  
안정적인 매장 데이터 백업 도구 입니다.  

---

### 🔹 동작 원리
- **스케줄 실행**: 설정한 시각(예: 매일 23:30)에 자동 백업 수행  
- **즉시 실행**: "백업 테스트(1회 실행)"으로 즉시 백업 가능  
- **대상 데이터 선택**:  
  - 매입/매출/신용카드: 오늘 기준 30일, 60일 등 기간 선택  
  - 그외 정보는 전체 백업
- **저장 및 전송**:  
  - 지정된 로컬 경로에 백업 저장  
  - FTP 서버(대리점/본사)로 자동 업로드 가능  
- **시스템 동작 옵션**: 백업 종료 후 시스템 자동 종료 가능  
- **자동 실행**: Windows 시작메뉴 등록 → 재부팅 후 자동 실행 지원  

---

### 📌 주요 기능
- **스케줄 기반 자동 백업**
- **1회 즉시 백업 실행**
- **선택적 데이터 백업 (30일/60/90일/180일 옵션 포함)**
- **로컬 저장 + FTP 업로드 지원**
- **백업 완료 후 시스템 종료 기능**

---

### 💻 시스템 요구 사항
- **운영체제 (OS)**  
  - Windows 7 SP1 (Service Pack 1) 이상  

- **필수 구성 요소**  
  - .NET Framework 4.8 (설치 프로그램에 포함 가능)  
  - MySQL 5.5 / 5.6 (서버 환경)

---

### 📖 사용 방법
1. 상단의 **다운로드** 버튼으로 설치 프로그램을 내려받습니다.  
2. 설치 후 **관리자 권한**으로 실행합니다.  
3. 백업할 **매입/매출/신용카드 항목**의 **기간 옵션**을 선택합니다. (나머지 데이터는 전체 백업)  
4. **저장 경로** 및 **FTP 업로드 여부**를 설정합니다.  
5. 백업 실행 시각을 지정하고 **[설정 저장]** 버튼을 누릅니다.  
6. 필요 시 **[백업 시작]** 또는 **[백업 테스트(1회 실행)]** 버튼으로 수동 실행 가능합니다.
